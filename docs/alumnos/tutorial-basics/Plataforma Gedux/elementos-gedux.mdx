# Elementos de la plataforma Gedux

## 1. Inicia sesión en Gedux

Como primer paso, debemos **iniciar sesión en la plataforma Gedux**. Si tienes dudas sobre cómo ingresar, puedes consultar el siguiente tutorial:  
[Accede al siguiente tutorial](./plataforma-pago)

## 2. Visualiza cada apartado

Una vez iniciada la sesión en la plataforma, podrás visualizar cada elemento con el que cuenta Gedux, como se muestra en la imagen:

![element](./img/ele1.jpg)

**Cada módulo cumple una función específica, las cuales se describen a continuación:**

1. **Información del usuario:** Muestra datos como el nombre, matrícula, asignaturas, estado académico, así como información personal. También incluye una opción para consultar calificaciones de materias culminadas.
2. **Próximo a pagar:** Este módulo ofrece la opción de pagar tu mensualidad pendiente a través del botón **Pagar**.  
   Si tienes dudas sobre cómo realizar este proceso, puedes acceder al siguiente tutorial.
3. **Adeudos:** Aquí se muestran los pagos pendientes.
4. **Pagos parciales:** En este apartado se enlistan todos los pagos realizados anteriormente.
5. **Otros pagos:** En este último apartado se reflejan otros pagos adicionales solicitados.

## 3. Menú de navegación

La plataforma Gedux incluye un menú de navegación con opciones como **Mis adeudos**, **Mis pagos** y **Calificaciones**.

![element](./img/ele2.jpg)

### Mis adeudos

Si seleccionamos la opción **Mis adeudos**, se mostrará la siguiente vista, en la que puedes observar si tienes algún pago pendiente por realizar:

![element](./img/ele3.jpg)

### Mis pagos

Al seleccionar **Mis pagos**, puedes ver los pagos previamente realizados y obtener el recibo correspondiente:

![element](./img/ele4.jpg)

Cuando eliges la mensualidad a pagar, aparecerán las siguientes opciones: **Descargas** y **Mis pagos**, las cuales permiten realizar estas acciones:

- **Descargas:** Te permite descargar el recibo generado en tu computadora.
- **Otros pagos:** Accede a consultar otras mensualidades previamente realizadas.

![element](./img/ele7.jpg)

### Calificaciones

En el módulo de **Calificaciones** puedes consultar tus resultados académicos por ciclos o materias culminadas, como se muestra en la imagen:

![element](./img/ele8.jpg)
